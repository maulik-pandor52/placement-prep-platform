    </p>

    <h2>✨ Key Features</h2>

    <h3>👤 User Features</h3>
    <ul>
        <li>Signup/Login system</li>
        <li>Initial aptitude + coding test</li>
        <li>Skill-based performance report</li>
        <li>Company recommendations</li>
        <li>Company-specific mock tests</li>
        <li>Final readiness report</li>
    </ul>

    <ul>
        <li>Manage users, questions, skills, and companies</li>
        <li>Track analytics and reports</li>
    </ul>

    <h2>🔥 Advanced Features</h2>
    <ul>

        <li><strong>Skill Gap Analyzer</strong></li>
        <li><strong>Gamified Learning System</strong></li>
        <li><strong>Industry Skill Tracker</strong></li>
    </ul>
